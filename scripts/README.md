# Scripts y guías operativas — CF Metal Pintura PRO

Esta carpeta contiene guías de instalación, configuración, verificación, migración y deploy.

## Archivos

```txt
scripts/
├── README.md
├── setup-local.md
├── deploy-frontend.md
├── deploy-backend.md
├── migrate-production.md
├── check-project.md
└── env-reference.md
```

## Uso recomendado

1. Leer `setup-local.md` para levantar el proyecto.
2. Revisar `env-reference.md` antes de configurar variables.
3. Usar `check-project.md` para validar módulos.
4. Usar `deploy-frontend.md` y `deploy-backend.md` para producción.
5. Usar `migrate-production.md` para migraciones seguras.
